系统流程图
```
flowchart TD
    A[客户端请求] --> B[Gin Router]
    B --> C{路由匹配}
    C --> D[JWT验证中间件]
    D --> E{验证成功?}
    E -->|否| F[返回401错误]
    E -->|是| G[API处理层]
    G --> H[Controller业务逻辑]
    H --> I[GORM数据操作]
    I --> J[MySQL数据库]
    J --> K[返回数据]
    K --> L[JSON响应]
    L --> M[客户端接收]
    
    N[配置文件] --> O[配置加载]
    O --> P[数据库连接初始化]
    P --> Q[服务启动]
```

实体关系图 (ERD)
```
erDiagram
    USER {
        int64 id PK
        string username
        string password
        string nickname
        string email
        string phone
        string avatar
        datetime create_time
        datetime update_time
    }
    
    ACTIVITY {
        int64 id PK
        int64 user_id FK
        string name
        string description
        string location
        datetime start_time
        datetime end_time
        int32 max_participants
        int32 if_delete
        datetime create_time
        datetime update_time
    }
    
    ACTIVITY_MEMBER {
        int64 id PK
        int64 activity_id FK
        int64 user_id FK
        datetime join_time
        int32 status
    }
    
    ACTIVITY_COMMENT {
        int64 id PK
        int64 activity_id FK
        int64 user_id FK
        string content
        datetime create_time
    }
    
    FRIEND {
        int64 id PK
        int64 user_id FK
        int64 friend_id FK
        int32 status
        datetime create_time
    }
    
    CHAT {
        int64 id PK
        int64 user_id_from FK
        int64 user_id_to FK
        string content
        datetime create_time
        int32 status_from
        int32 status_to
    }
    
    FILE {
        int64 id PK
        int64 user_id FK
        string file_name
        string file_type
        int64 file_size
        string file_path
        int64 link_file_id FK
        datetime create_time
    }
    
    USER ||--o{ ACTIVITY : creates
    USER ||--o{ ACTIVITY_MEMBER : joins
    USER ||--o{ ACTIVITY_COMMENT : comments
    USER ||--o{ FRIEND : has_friend
    USER ||--o{ CHAT : sends_from
    USER ||--o{ CHAT : receives_to
    USER ||--o{ FILE : uploads
    ACTIVITY ||--o{ ACTIVITY_MEMBER : has_members
    ACTIVITY ||--o{ ACTIVITY_COMMENT : has_comments
    FILE ||--o| FILE : links_to
```
